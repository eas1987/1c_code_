Использование рекурсивного запроса для создания таблицы иерархии структуры предприятия. 
Основная функция: "ПолучитьШтатнуюРасстановку()" для формирования таблицы + рекурсивные функции для работы с полученной таблицей. 

Описание задачи:
Структура предприятия задана в виде дерева, где группы - это подразделения, а элементы - должности. 
Глубина иерархии для разных подразделений может отличаться. На каждом уровне иерархии может быть одна руководящая должность или не быть ни одной такой должности. 
Необходимо получить непосредственного руководителя сотрудника или иерархию всех руководителей сотрудника.
